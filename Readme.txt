To run the program:

1 - Navigate to the directory in you terminal that has the main.c and the TaskSpec.txt files

2 - using the terminal type in: "gcc -o main main.c" to compile the c code.

3 - Then type in: "./main TaskSpec.txt" to run the program.

4 - To view the output, after the program has exited, type: "nano Output.txt" in the terminal
